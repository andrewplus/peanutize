About

In late 2021, the Peanuts Movie character creator at peanutizeme.com went offline. Using a mix of Wayback Machine snapshots and a few carefully recreated missing assets, this project aims to keep the Peanutize website alive, while also providing several bugfixes and quality-of-life improvements. The source code can be found here.

Changes

  • All limited-time seasonal customizations have been re-added and are available year-round
  • The mobile layout has been completely re-built due to a lack of archived assets
  • A transparent PNG of your character can now be downloaded by clicking the "Character Only" download option at the end screen
  • A "Save" button has been added to the "My Characters" gallery
  • The "New" button on the My Characters page now allows the user to choose between creating a human or dog, instead of jumping straight into the human character creator
  • The trailer video now always uses YouTube to save on bandwidth
  • The default clothing for new human characters has been altered to be more neutral
  • The recent characters gallery on the home page now loads from a staticfile instead of the dead API
  • Legal information in footer is now fetched from local file instead of the defunct foxprivacy.com
  • Google Analytics, other tracking removed
  • All locales aside from United States and Japan have been removed from the footer dropdown menu as they are currently not implemented in this archive
  • Sharing to Facebook and Twitter has been disabled
  • Fixed a bug that prevented dog characters from being edited after being created
  • Odd positioning of certain accessory color variations fixed
  • Some missing assets have been recreated as accurately as possible using screenshots due to being absent from the Wayback Machine archive
