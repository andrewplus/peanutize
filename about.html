About

In late 2021, the Peanuts Movie character creator at peanutizeme.com went offline. Using a mix of Wayback Machine snapshots and carefully recreated assets, this project aims to keep the project alive, while providing several bugfixes and quality-of-life improvements.

Feature Changes

  • All limited-time seasonal customizations have been re-added year-round
  • Created characters can now be downloaded as a transparent PNG from the save screen
  • The "New" button on the My Characters page has been changed to lead to the species select screen instead of directly to the human gender select screen
  • Odd positioning of certain accessory color variations fixed
  • The default male/female clothing has been altered to not include seasonal clothing
  • Trailer now only loads via YouTube to reduce bandwidth
  • All locales aside from United States have been removed from the footer dropdown menu
  • Sharing is disabled
  • The mobile version is currently not supported

Technical Changes

  • Several lost assets have been recreated as accurately as possible.
  • Gallery changed to load from static "characters/random.json" file instead of the dead "peanutize.com/characters/random" API
  • Google Analytics, other tracking removed
  • Legal information in footer is now fetched from local file instead of the defunct foxprivacy.com
  • Footer text changed to reflect the nature of this fan archive
