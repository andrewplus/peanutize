About

In late 2021, the Peanuts Movie character creator at peanutizeme.com went offline. Using a mix of Wayback Machine snapshots and carefully recreated assets, this project aims to keep the website alive, while providing several bugfixes and quality-of-life improvements.

Feature Changes

  • All limited-time seasonal customizations have been re-added and are active year-round
  • Created characters can now be saved as a transparent PNG by clicking the new "Character Only" option on the save screen
  • The "New" button on the My Characters page now allows the user to choose between creating a human or dog, instead of jumping straight into the human character creator
  • The default clothing for new human characters has been altered to exclude seasonal clothing
  • The trailer video now only uses YouTube instead of video.js.
  • All locales aside from United States have been removed from the footer dropdown menu
  • Sharing to Facebook and Twitter has been disabled
  • The mobile layout is currently not supported due to a lack of archived assets

Bugfixes

  • Fixed a bug that prevented dog characters from being edited after being created
  • Odd positioning of certain accessory color variations fixed

Technical Changes

  • Several missing assets have been recreated as accurately as possible due to being absent from the Wayback Machine archive
  • Recent characters gallery now loads from static "characters/random.json" file instead of the dead "peanutize.com/characters/random" API
  • Legal information in footer is now fetched from local file instead of the defunct foxprivacy.com
  • Google Analytics, other tracking removed
